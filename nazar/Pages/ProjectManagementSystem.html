Project Management System

Client Desktop Application

Project details

Description
It is challenging for a company to handle the daily tasks without having a structured and an easy way to get an overview over the workload. Therefore, the goal of this project is to build a project management system that is user friendly, efficient, and easy to maintain. This project management system is not just a simple stand-alone application designed specifically only for the client’s needs. It is an application that can be easily adapted to multiple companies, making it affordable to maintain and update. If the client wants extra functionalities or changes in aspects of the system, those changes can be adapted to the existing source code of the system. The system has been analysed, designed, implemented and tested in a group work with people from different cultural backgrounds. The system have been deigned using Waterfall Methodology.
